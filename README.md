# Maven Electronics Sales Pipeline Analysis (Excel Project)

## Project Overview
This project was completed as part of a sales analytics initiative at Maven Electronics, a global electronics retailer specializing in computer components for business clients.

The objective was to analyze CRM sales opportunity data to identify trends, evaluate performance, and provide actionable insights to improve sales effectiveness.

---

## Business Objective
- Analyze sales pipeline performance  
- Evaluate sales agent productivity  
- Identify top-performing products  
- Understand deal progression across stages  
- Provide insights to improve sales outcomes  

---

## Tools and Skills Used
- Microsoft Excel  
- Pivot Tables and Pivot Charts  
- Data Cleaning and Transformation  
- Calculated Fields  
- Data Aggregation and Analysis  

---

## Dataset Summary
- CRM data including sales agents, products, accounts, and deal stages  
- Total Sales: 10,005,534  
- Total Opportunities: 3,512

# REPORT 1: SALES AGENT PERFORMANCE

## Objective
Evaluate individual sales agent performance based on sales, opportunities, and win rate.

## Key Findings
- Top agents consistently achieve win rates above 65%  
- Performance varies significantly across agents  
- High opportunity volume does not always guarantee higher win rates
